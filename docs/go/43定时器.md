### 2.定时器配合goroutine
- timer.NewTicker和JS的setInterval类似
```go
package main

import (
	"fmt"
	"time"
)

func main(){
	// 1.获取ticker对象
	ticker := time.NewTicker(1 * time.Second)

	i := 0
	go func() {
		for{
			i++
			fmt.Println(<-ticker.C)
			if i == 5{
				// 停止
				ticker.Stop()
			}
		}
	}()

	// 为了不让协程退出
	for{}
}
```
